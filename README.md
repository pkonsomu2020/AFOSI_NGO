# AFOSI - Action for Sustainability Initiative

![AFOSI Logo](public/afosi_logo.png)

## About AFOSI

**Action for Sustainability Initiative (AFOSI)** is a Kenya-based non-profit organization founded in 2012, committed to improving the quality of life of children, youth, and women (aged 10-35) towards the attainment of key Sustainable Development Goals (SDGs).

AFOSI uses human-centered design and a rights-based approach in programming, working across five core thematic pillars:

- 🏥 **Health** - Promoting well-being and access to healthcare
- 📚 **Education** - Empowering through knowledge and skills
- 🌍 **Environment** - Advancing climate justice and sustainability
- 💼 **Livelihoods** - Creating economic opportunities
- 👥 **Leadership & Governance** - Building capacity and advocacy

## Our Mission

To advance sustainable development, social justice, gender equality, health, and inclusive community empowerment through innovative, community-driven solutions.

## Our Approach

AFOSI adopts a hybrid implementation model that combines:
- **Grassroots NGO reach** - Community trust and local engagement
- **Social Enterprise innovation** - Technology-enabled solutions and scalable interventions

We design, deploy, and partner around digital platforms and tools that translate research and community insights into practical, measurable impact.

## Our Programs

### 1. We Lead Project
Funded by HIVOS East Africa through the Center for Adolescence (CSA), this program empowers young women facing multiple barriers in society, focusing on Sexual and Reproductive Health Rights (SRHR) through leadership and social justice.

**Impact:** 500+ young women empowered

[Learn More →](/programs/we-lead)

### 2. Robotics & Creative Coding
In collaboration with STEM IMPACT CENTER, we introduce youth to creative coding and robotics, developing critical thinking and problem-solving skills for the digital economy.

**Impact:** 300+ youth trained over 12 months

[Learn More →](/programs/robotics-coding)

### 3. Blancosy
An eco-entrepreneurship initiative in Nairobi's informal settlements, reducing plastic waste through innovative recycling while creating economic opportunities.

**Impact:** 200+ youth engaged in green jobs

[Visit Website →](https://blancosy.co.ke)

### 4. Kiongozi Platform
An emerging leaders platform providing mentorship, capacity-building, and leadership training for grassroots youth, women, and persons with disabilities.

**Impact:** 1000+ emerging leaders supported

[Visit Platform →](https://www.kiongozi.ke)

## Development

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <repository-url>

# Navigate to project directory
cd afosi-website

# Install dependencies
npm install

# Start development server
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for Production

```bash
# Create production build
npm run build

# Preview production build
npm run preview
```
